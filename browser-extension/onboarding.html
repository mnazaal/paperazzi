pzi capture — setup

Configure the extension so it can talk to your local pzi server.

Before you start

  1. Install pzi: pip install 'pzi @ git+https://github.com/mnazaal/pzi.git'
  2. Run: pzi init --setup --bib ~/bibs/main.bib — creates ~/.config/pzi/config.toml
  3. Start the server: pzi server (it also runs the translation-server)

Extension configuration

Next steps

  1. Navigate to a paper page (e.g. arXiv abstract, publisher page)
  2. Click the pzi toolbar icon → Capture current page
  3. Or right-click any paper link → Save to pzi

If capture fails

  • Is the server running? Start it with pzi server
  • Is the token correct? Check token in ~/.config/pzi/config.toml
  • Firewall? pzi listens on 127.0.0.1:8765 by default.
